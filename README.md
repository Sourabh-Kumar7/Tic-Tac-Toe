# Tic-Tac-Toe
Its a simple TIC-TAC-TOE GAME implemented in CPP.

## Installation
In case you don't have git
`
sudo apt-get install git
`
Clone the repo
`
git clone https://github.com/beyourwild7/Tic-Tac-Toe.git
`
Open the directory
`
cd Tic-Tac-Toe
`
## Usage
### Compilation
`
g++ Game.cpp
`
### Run
`
./a.out
`
